# SQL Portfolio – Customer & Sales Analysis
This project contains SQL queries for analyzing customer behavior, product performance, and revenue insights using a fictional sales database.

---

## 📊 Dataset
- Customers
- Products
- Orders

---

## 🔧 Skills Demonstrated
- Data cleaning and table creation
- SQL joins (INNER JOIN)
- Aggregations (SUM, COUNT, AVG)
- Filtering and grouping
- CTEs (Common Table Expressions)
- Window functions (RANK)
- Business insights generation

---

## 📈 Key Analyses

### 1. Revenue Analysis
- Revenue by city
- Revenue by product
- Revenue by category

### 2. Customer Analysis
- Top customers by revenue
- Customers above average spending
- Customers with no orders

### 3. Advanced Analysis
- Top 3 customers per city
- Ranking customers using window functions
- Variety of products purchased per customer

---

## 🚀 Purpose
This project was built as part of my Data Analyst learning journey to demonstrate SQL skills applied to real-world business problems.
